# RD Engine: Game Engine made in Monogame

I made this game engine on top of the Monogame C# library, trying to implement many of the features present in off-the-shelf engines.

It is not perfect, and is mostly a proof of concept and test of my skills as a current student. I based most of my design decisions in my knowledge of Unity and,
while it doesn't even have a GUI, I'm pretty proud of the result for the time I spent on it.

As this was mostly a learning experience, no AI was used during its development.

This project also contains a game I made with the engine: The Phantom Furniture Frenzy, which I made, along with parts of the engine, over the course of two weeks.
